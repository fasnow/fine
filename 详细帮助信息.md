### 1️⃣  **资产测绘模块**

- **Host 可自定义**（需保证所有查询和用户信息接口响应与官网一致）
- **FOFA**：根据 API 等级配置查询字段，超出权限接口会提示错误  
- **Hunter**：所有表格字段均可勾选，无等级限制、限制境外IP访问  
- **Quake**：所有表格字段均可勾选，仅返回当前等级数据，非会员请勿勾选全部数据  
- **ZoomEye**：字段无级别限制，仅返回当前等级数据，注册积分不可用于API查询  
- **Shodan**：暂无特殊说明

### 2️⃣ **ICP 模块相关**

- 📝 **OCR 脚本使用提醒**  
    - ⚠️ 若不熟悉 IDE 运行配置，不建议直接运行 OCR 脚本  
- ⏩ **批量查询 & 代理池建议**  
    - 🚩 请务必使用 **高质量代理池**（勿用免费代理，质量不可控，影响体验），推荐购买临时代理池  
    - 💡 常见 IP 切换方式：
        - 每秒切换
        - 每分钟切换
        - 每次请求切换 **（推荐，任务量大更适合）**
    - ⏱️ 不同代理池有请求速率限制，详情请参考购买说明
    - ⚙️ 合理配置 “并发量” 和 “并发单位时间（ns）”，以适配代理池策略
    - 🔄 批量查询自带重试机制，不建议设置过大，异常次数过多时应中止，可断点续查无需重新开始

### 4️⃣ **小程序敏感信息模块**

- 📁 目录设置点【帮我选择】即可，无需手动配置
- ⏳ 部分信息查询不到系因三方接口偶发异常，异常时会自动5分钟后重试，欢迎提供更优方案
- 🧵 默认线程数为 2 倍 CPU 核心数，保证系统资源平衡，一般无需调整
- 非完整反编译，不可导入微信开发者工具，https://github.com/Ackites/KillWxapkg 提供完整结构
